# Nonlinear dimensionality reduction
This project is part of my master's thesis, under the supervision of Dr. Enjie GHORBEL and Pr. Faouzi GHORBEL (Data Science Master @ENSI, 2026). 

## First part of the project
The first part of this project consists of understanding the theoretical foundations, implementing, and analyzing the limitations of state-of-the-art methods for linear and nonlinear dimensionality reduction.

## Note
This repository is organized into several folders, each covering a specific concept along with its Python implementation and result interpretation. Each folder also includes a presentation detailing the underlying mathematics. If you are interested in understanding the mathematics behind these concepts, feel free to explore these notes. Note that DR means Dimensionality Reduction.
